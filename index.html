Panorama Generator
This is a script that generates a panorama image from multiple related images.


The user takes several images of a certain location, insert their path on and the program creates a panorama image from thos images.


The images must have common area for it to work.


The scirpt uses algorithms such as SIFT, RANSAC, homography calculations and image registration.


Tools:Pycharm.
Languages:Python.
Packages:opencv, matplotlib, numpy.
Project Link:IPCV-Project Github.
